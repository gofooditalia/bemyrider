──────────────────────────────────────

Fast Path (Logged-In User):
┌─────────────────────────────────────────────┐
│ SharedPrefs Check (5ms) → Navigate Home (10ms)│
├─────────────────────────────────────────────┤
│ [0ms]  [15ms]                               │

In Background (Non-Blocking):
  getSiteSettingData (1-3s)  ────────────────────────────┐
  saveOfflineData (500ms)    ─────────┐                  │
  [These run AFTER user enters Home]   │                  │
                                       ↓                  ↓
                              [User is already using the app!]

User Experience: ⚡ Home opens in 100ms, data loads silently
Performance: Parallel execution (home + background tasks)
Reliability: Home opens even if getSiteSettingData fails (cached data)
User Acquisition: Instant gratification, higher retention
